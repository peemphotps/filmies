# Design originals

The original Claude Design handoff prototype that the native SwiftUI app was
built from. These are reference artifacts — they are **not** part of the Xcode
build.

- `Filmies.html` — runnable browser prototype entry point
- `camera.jsx` — main camera app (viewfinder + controls)
- `viewfinder.jsx` — live CSS-filter viewfinder + canvas capture baking
- `recipes.jsx` — the 13-stock film recipe definitions (ported to `FilmRecipe.swift`)
- `pickers.jsx` — film picker variants (dial / wheel / sheet)
- `tweaks-panel.jsx` — exposure / grain adjustment panel
- `ios-frame.jsx` — the prototype's fake iOS device chrome
- `_chips.png`, `_chips2.png` — design reference images
